### Getting Started ###

Clone the application with:

* HTTPS `git clone <https-link>` 
* SSH `git clone <ssh-link>`

### Dependencies ###

* nodejs v6.9.1
* npm 4.0.2

Run `npm install -g cordova ionic` to install latest cordova and ionic

### Running The Server ###

* Run `ionic serve --nolivereload`
